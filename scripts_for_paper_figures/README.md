## This folder contains 3 important scripts:
- `Example_simulation_NLI_vs_PC1_comparison.m`: A script for displaying the relationship between PC1 derived from white noise analysis and NLI from isoresponse phase of the experiment. 
- `Popfigures_for_Elife_rebuttal_manuscript.m`: A script for generating key figures for the De and Horwitz, 2021 paper. 
- `Popfigures_Elife_for_response_to_reviewers_comments.m`: A script for generating some of the figures for response to reviewer's comments.
